# Stephan Gilles Portfolio

Angular-Portfolio-Projekt für die Developer Akademie.

## Start lokal

```bash
npm install
npm start
```

Danach im Browser öffnen:

```text
http://localhost:4200
```

## Windows Schnellstart

```text
start.bat
```

## Build prüfen

```bash
npm run build
```

## Tech-Stack

- Angular 17
- TypeScript
- SCSS
- Responsive Design
- Git / GitHub

## Projektstruktur

```text
src/app/core       Header, Footer, i18n, Scroll-Service
src/app/pages      Home, Impressum, Datenschutz
src/app/sections   Hero, About, Skills, Projects, References, Contact
src/assets         Bilder, Icons, Übersetzungen
```

## Abgabehinweis

Die ZIP enthält den Quellcode ohne `node_modules`. Die Abhängigkeiten werden über `npm install` installiert.
